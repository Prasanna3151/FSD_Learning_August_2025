CSS

What is CSS?

CSS is giving style to a HTML page. HTML is just the sceleton of a webpage. Using CSS we can add colors, positioning and basically it is used for styling. CSS - CASCADING STYLE SHEETS

CSS has three heirarchies:

  • User Agent stylesheet - Present defaultly in the browser
  • User stylesheet - User can change temporarily in the browser
  • Author Stylesheet - This is what we are going to learn and write code.

CSS Selectors

CSS selectors are used to style a particular block of HTML based on what we select.

There are 4 types of selectors:

  1. Universal selectors - uses "*", applies style to all the elements.
  2. Type Selectors - uses element name, applies style based on elements.
  3. Class selectors - uses ".", applies style based on class.
  4. ID selectors - uses "#", applies style based on ID. Each element should have a unique ID.
