UNITS

Types of units:

  • px - pixels, gives absolute length.
  • em - Relative to parent's font size
  • rem - relative to the root element font size.(Defaut root element is 16px.)
  • vw - 1% of Viewport's width
  • vh - 1% of Viewport's height.
  • ch - Width of 0 characters
  • % - Percentage, usually related to parent values.

Use Cases:(writen based on preference)

Width/Height

  1. % - to be relative to parent
  2. vw/vh
  3. ch
  4. rem
  5. px

Margins / Padding

  1. rem
  2. em
  3. px

Borders

  1. px
  2. rem/em

Font size

  1. rem - best
  2. em - scale to parent
  3. px - last option
